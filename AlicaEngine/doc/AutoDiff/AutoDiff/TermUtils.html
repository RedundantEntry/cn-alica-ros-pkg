Compile Method

Creates a compiled representation of a given term that allows efficient evaluation of the value/gradient.

Syntax

public static ICompiledTerm Compile (this Term term, params Variable[] variables)

Parameters

term
The term to compile.
variables
The variables contained in the term.

Returns

A compiled representation of term that assigns values to variables in the same order as in variables

Remarks

The order of the variables in variables is important. Each call to ICompiledTerm.Evaluate or ICompiledTerm.Differentiate receives an array of numbers representing the point of evaluation. The i'th number in this array corresponds to the i'th variable in variables.

Differentiate Method

Computes the gradient of the function represented by a given term at a given point.

Syntax

public static double[] Differentiate (this Term term, Variable[] variables, double[] point)

Parameters

term
The term representing the function to differentiate.
variables
The variables used in term.
point
The values assigned to the variables in variables

Returns

The gradient of the function represented by term at the point represented by variables and point.

Remarks

The i'th value in point corresponds to the i'th variable in variables. In addition, the i'th value in the resulting array is the partial derivative with respect to the i'th variable in variables.
